# Résistance Anarchiste

Plateforme web de résistance anarchiste proposant :

- **Manifeste** : Texte de résistance et d'émancipation
- **Galerie de visuels** : Collection d'affiches militantes
- **Réseau CNT-AIT** : Contacts des collectifs locaux
- **Liens sociaux** : Coordination sur Telegram, Twitter, etc.

Cette application combine un frontend React moderne avec un backend Flask léger, optimisée pour un déploiement sur YunoHost.

## Fonctionnalités

- Interface responsive et accessible
- Galerie d'images avec partage social
- Carte interactive des collectifs (à venir)
- Système de navigation fluide
- Optimisée pour mobile

## Sécurité

L'application est configurée avec des mesures de sécurité appropriées pour YunoHost :
- Utilisateur système dédié
- Permissions restreintes
- Configuration systemd sécurisée
- Proxy NGINX optimisé
